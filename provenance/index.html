How it works

1

Point it at a directory

Install. Run signal-provenance init. It starts watching. Every file creation, modification, and deletion is perceived.

2

Hash-chain builds automatically

Every observation is SHA-256 hashed and chained to the previous one. Tamper with any entry, and every subsequent hash breaks. Math, not policy.

3

Verify and export

One command to verify: signal-provenance verify. Export PDF or JSON compliance reports covering FDA, SOX, CMMC, HIPAA, ISO, EU AI Act.

What you get

Continuous file witnessing

Every file change perceived and recorded. Content hash, timestamp, actor, size. Automatic.
