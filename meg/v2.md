# MEG MVP Architecture - v2 (Expanded Capabilities)

14 sections. Added deep AI/ML coverage (6 services) and full security architecture with incident response.

## Changes from v1

### Added
- **AI/ML Intelligence Layer** - Expanded from overview to 6 distinct ML services:
  - 5.1 Demand/Supply Forecasting (XGBoost/LSTM/Prophet ensemble, feature sets, accuracy targets, training pipeline)
  - 5.2 Dynamic Pricing Optimization (LP-based, time-of-use, incentive shaping, guardrails)
  - 5.3 Anomaly Detection (Isolation Forest, LSTM autoencoder, DBSCAN, response pipeline with scoring tiers)
  - 5.4 Battery Storage Optimization (MILP via OR-Tools, 24h rolling horizon)
  - 5.5 Grid Load Balancing (congestion management, voltage regulation, frequency support, OpenADR 2.0b)
  - 5.6 Trade Recommendation Engine (rule-based MVP, evolving to RL)
  - 5.7 ML Operations (MLflow, drift detection, validation gates, rollback)
- **Security Architecture** - Full section:
  - Edge security (X.509, ed25519, secure boot, TPM)
  - Transport security (mTLS, JWT, Redis AUTH)
  - Application security (OAuth 2.0 + PKCE, RBAC, Pydantic, rate limiting, CSP)
  - Smart contract security (OpenZeppelin patterns, pausability, proxy upgrades, formal audit)
  - Incident response (SEV-1/2/3 definitions, response timelines)

### Modified
- IoT section expanded with data validation rules table
- Matching engine expanded with forecast-informed bidding
- System architecture diagram updated with all 6 ML services
