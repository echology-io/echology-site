Signal, Surface, Spec, and Syncro are different applications of the same processing pipeline. Deterministic classification before probabilistic inference. Provenance at every step. Structure revealed, never fabricated.

Processing Pipeline

How information flows through the system

Every document, every query, every analysis follows the same path. The layers are always in this order. No shortcuts.

Layer 1

Perception

Perceive, classify, embed

Documents enter the system and are classified deterministically by type, structure, and content. No language model at this stage. Files are hashed for integrity, indexed with source labels, and embedded for semantic retrieval. Multi-server ingestion means distributed environments merge into a single view.

File classification Content hashing Semantic embedding Multi-server ingestion
Layer 2

Verification

Verify, audit, trace

Every classification, every score, every output is verified against the source material. An audit ledger tracks what was processed, when, by which model, and with what confidence. Quality certification ensures outputs meet threshold before they reach a human. Nothing is asserted without provenance.

Audit ledger Quality scoring Source tracing Confidence thresholds
Layer 3

Retrieval

Retrieve, reason, report

Verified content is retrieved semantically and structured for analysis. AI models (local or external, your choice) reason over the structured data. Reports are generated with full provenance chains -- every statement traced back to source documents. The deterministic layer provides the facts; the AI layer provides the analysis.

Semantic retrieval Multi-model reasoning Provenance chains Structured reporting
Foundation

Decomposition

Deterministic NLP. Zero dependencies.

The base layer that every other layer depends on. Text classification, entity extraction, structural analysis -- all without a language model. No API calls. No inference costs. No hallucination risk. This is the layer that makes the entire system deterministic-first. It runs on any hardware, offline, with zero external dependencies.

Zero dependencies Offline capable Text classification Entity extraction

Design principles

Deterministic before probabilistic

Classification, scoring, gap detection, and dependency mapping use deterministic algorithms. AI models layer on top for generation and analysis -- never in place of the deterministic foundation.

On your hardware

The system runs on your machines. Your data never leaves your network. No cloud dependencies. No SaaS subscriptions. No vendor lock-in. You own everything the system produces.

Provenance at every step

Every output traces back to its source. Every classification has a reason. Every score has a formula. Every report has a chain of evidence. Nothing is asserted without proof.

AI architecture

You choose which models. We provide the architecture that makes them useful.

Tier 1

Deterministic

Classification, scoring, gap detection, dependency mapping. Zero hallucination. Zero cost. Zero latency beyond computation. This layer never changes regardless of which AI models are available.

Tier 2

Local AI

Models fine-tuned on your approved corpus. Run on your hardware. Optimized for narrow tasks: operational Q&A, domain inference, context-aware analysis. You own the model weights.

Tier 3

External AI

Claude, GPT, or any API for high-quality generation when you choose. All outputs scored deterministically against ground truth. The system suggests the best model for each task. No vendor lock-in.
